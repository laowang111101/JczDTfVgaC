## 项目概述

本项目是基于Spring Boot框架开发的一款银行业务系统，旨在为管理员、用户和银行员工三种角色提供安全、高效、便捷的金融服务。系统通过信息化手段，实现账户管理、交易处理、客户服务和系统管理等功能模块，满足不同角色的操作需求。

## 技术栈

- 后端：Spring Boot、Java
- 前端：HTML/CSS/JavaScript（技术栈未明确，可根据实际项目调整）
- 数据库：MySQL（或其他关系型数据库）

## 功能模块

### 系统角色

- 管理员
- 银行员工
- 用户

### 管理员功能需求

- 系统管理：系统首页、个人中心、公告通知管理
- 用户管理：账户管理
- 员工管理：账户管理、角色分配、权限管理
- 业务数据管理：账户信息管理、转账信息管理、存款信息管理、取款信息管理、理财产品管理
- 人力资源管理：员工薪资管理、考勤打卡管理

### 员工功能需求

- 系统首页
- 业务办理：用户业务办理、业务审核管理
- 人力资源管理：员工薪资查询、考勤打卡

### 用户功能需求

- 系统首页
- 个人中心
- 账户信息管理
- 业务数据管理：转账信息、存款信息、取款信息、理财产品
- 系统维护：公告通知管理
